# 🎬 AnimeFetcher Bot

![AnimeFetcher Banner](C:\Users\hhp\.gemini\antigravity\brain\a502c61d-f27a-4a73-ac7d-cee5d8c7bbf7\anime_fetcher_banner_png_1774105278693.png)

A powerful, automated Telegram bot designed to forward anime media from source channels, generate stunning banners, and manage your watchlist with ease.

## 🚀 Features
- **Auto-Forwarding**: Seamlessly forwards media from your favorite source channels.
- **Banner Generation**: Creates high-quality anime information banners using AniList API.
- **Watchlist Management**: Track your favorite anime and get notifications for new episodes.
- **Port Binding**: Built-in health check server for 24/7 deployment on Render.

## 📜 Rules & Usage
1. **Personal Use**: This bot is intended for personal use and small community management.
2. **Channel Ownership**: Ensure you have permission to forward media from the source channels you configure.
3. **No Spamming**: Avoid using the bot to spam Telegram channels or groups.

## ⚙️ Compatibility
- **Platform**: Python 3.8+
- **Environment**: Local PC (Windows/Linux) or Cloud Platforms (Render, Heroku, etc.).
- **Libraries**: Telethon (UserBot), PTB (Bot interface), Httpx (API calls), Pillow (Banner processing).

### How to Run Locally
1. Install dependencies: `pip install -r requirements.txt`
2. Configure `.env` with your `API_ID`, `API_HASH`, and `BOT_TOKEN`.
3. Run: `python bot.py`
